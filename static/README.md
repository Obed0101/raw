## ?? ¿Qué es Vivero Dashboard?

Es un template de SvelteKit ultra-optimizado para crear dashboards modernos con:

- **Código mínimo**: Menos de 200 líneas de código total
- **Configuración simple**: Solo editar .env
- **Tema oscuro**: Con efecto glassmorphism moderno
- **Melt UI**: Componentes accesibles y optimizados
- **Responsive**: Funciona en móvil y desktop
